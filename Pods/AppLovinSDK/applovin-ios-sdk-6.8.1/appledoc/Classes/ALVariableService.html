ALVariableService Class Reference

Inherits from NSObject

Overview

This service allows for retrieval of variables pre-defined on AppLovin’s dashboard.

  delegate

Register a delegate to be notified when the latest variables are retrieved from the server. The initial set of variables will be returned after the AppLovin SDK finishes initializing.
